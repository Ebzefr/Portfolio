Lofis Restaurant

Ebenezer Obi
March 2024

This full-stack web application was designed and developed for a fictional themed takeaway restaurant. Leveraging ASP.NET with Razor Pages, the project integrates C#, HTML/CSS, and JavaScript to build a seamless and interactive user experience for both customers and administrators. The backend is supported by an SQL database to handle dynamic content such as menus and orders. An intuitive admin panel was developed to enable restaurant staff to efficiently manage menu items and track orders in real-time.
The application was thoroughly tested and deployed on a Plesk Server, showcasing end-to-end skills in front-end development, server-side programming, database management, and deployment.

Problems

  • Need for an easy-to-use online ordering system for a takeaway restaurant.
  • Managing dynamic menu content efficiently.
  • Ensuring smooth order processing and status updates.
  • Providing restaurant staff with an intuitive admin interface to manage orders and menus.

Goals

  • Develop a responsive web app for customers to browse menu and place orders.
  • Implement a secure SQL database to handle dynamic menu and order data.
  • Build an admin dashboard for staff to update menus and track orders in real-time.
